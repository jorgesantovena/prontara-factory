# Smoke test del flujo chat → PR

Este fichero se ha creado desde el chat de Prontara Factory en producción
usando la tool commit_to_github_pr el 5 de mayo de 2026.

Si lo estás leyendo en main, significa que:
1. El chat ha podido generar un PR contra GitHub vía API.
2. Tú has revisado el PR y lo has merged.
3. Vercel ha redesplegado automáticamente.

El loop chat → PR → deploy funciona end-to-end.
